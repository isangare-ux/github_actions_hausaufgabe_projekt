# GitHub Actions Hausaufgabe

Dieses Projekt ist ein kleines Bash Projekt fuer GitHub Actions.

Aufgabe der Teilnehmenden:
1. Projekt in ein eigenes GitHub Repository hochladen.
2. Drei GitHub Actions Workflows schreiben.
3. Workflows ausfuehren.
4. Kurz erklaeren, was jede Pipeline macht.

Hinweis:
Der Ordner MUSTERLOESUNG_workflows enthaelt fertige Beispiel Workflows fuer die Lehrkraft.
Wenn die Teilnehmenden die Workflows selbst schreiben sollen, kann dieser Ordner vor der Ausgabe geloescht werden.
